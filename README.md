# Interior Harmony Website

A simple static interior design landing page with a white and light yellow palette. The site includes sections for modular kitchen design, WPC louvers, and PVC panel styling.

## Files

- `index.html` — main landing page
- `styles.css` — site styling

## GitHub Pages Deployment

1. Create a GitHub repository and push these files.
2. In repository settings, enable GitHub Pages from the `main` branch.
3. Set the site folder to `/ (root)`.
4. Visit the published URL after a few minutes.

## Notes

- Replace the placeholder image blocks in `index.html` with your actual pictures.
- Use the section IDs `#services`, `#gallery`, and `#contact` for quick navigation.
